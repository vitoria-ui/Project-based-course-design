这是一个 **学生信息管理系统**，采用 C++ 编写，核心功能是对学生的姓名和成绩进行全面的管理。以下是其主要特点和功能的简单描述：

### 核心功能
1. **登录验证**：
   - 系统启动时需要输入管理员密码（默认密码为 "admin"），验证通过后才能进入主功能菜单。

2. **学生信息操作**：
   - **添加学生**：支持单个添加或批量添加学生信息（姓名 + 成绩），批量添加时可一次输入多个学生，且会校验成绩是否在 0~100 之间。
   - **查看学生**：展示所有已存储的学生姓名和成绩。
   - **修改学生**：根据姓名查找学生，并修改其成绩。
   - **删除学生**：根据姓名删除学生信息，删除后后续学生信息会自动前移填补空缺。

3. **排序功能**：
   - 支持按 **成绩** 排序（从高到低或从低到高）。
   - 支持按 **姓名** 排序（升序 A-Z 或降序 Z-A）。

4. **查询功能**：
   - **姓名模糊查询**：输入部分字符，查找姓名中包含该字符的学生。
   - **成绩精确查询**：输入具体分数，查找成绩等于该分数的学生。
   - **成绩范围查询**：输入最低分和最高分，查找成绩在该范围内的学生。

5. **成绩统计**：
   - 自动计算并显示所有学生的 **平均分**（保留 1 位小数）、**最高分** 和 **最低分**。

6. **成绩格式化输出**：
   - 将学生成绩按分数段划分等级（A 优秀、B 良好、C 中等、D 及格、E 不及格），并显示等级描述。

7. **数据持久化**：
   - 支持将学生信息保存到 `students.txt` 文件（格式：姓名 成绩）。
   - 系统启动时会自动加载该文件中的历史数据，无需重新输入。

8. **清空功能**：
   - 可一键清空所有学生信息（需确认操作，避免误删）。


### 系统特点
- **界面简洁**：通过控制台菜单展示功能，操作直观，输入数字即可选择对应功能。
- **数据校验**：批量添加时校验成绩范围（0~100），避免无效数据。
- **用户体验**：操作后会给出明确反馈（如“添加成功”“未找到学生”），并在关键操作（如清空）时要求确认。
- **容量限制**：采用数组存储学生信息，最多支持 100 名学生。

### 使用流程
1. 输入密码登录系统。
2. 从主菜单选择功能（如添加、查看、修改学生等）。
3. 按提示输入相关信息，完成操作。
4. 退出系统时，数据会自动保存（若之前已保存过），下次启动可直接加载。
